# MuseumOfCurios

## Overview
MuseumOfCurios is a console-based museum simulator that allows users to browse and examine a collection of unique items known as "curios." The program simulates the experience of exploring a curated exhibit by letting users select items and view detailed descriptions.

This project was developed as part of a C# assignment to demonstrate object-oriented programming concepts, including classes, inheritance, and structured program design.

---

## Features
- View a complete list of available curios
- Select a curio to examine
- Display detailed descriptions of each item
- Continuous exploration through a looping interface
- Exit the program at any time

---

## Technologies Used
- C#
- .NET (Console Application)
- Object-Oriented Programming principles

---

## Project Structure
- **Program**: Entry point of the application
- **LibraryApp**: Handles user interaction and program flow
- **CurioCatalogue**: Stores and retrieves curio data
- **Curio (abstract class)**: Base structure for all curios
- **Curio subclasses**: Specific implementations of different curios
- **RarityLevel (enum)**: Defines rarity categories for curios
